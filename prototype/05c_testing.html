Vulnerability: Cross-Site Scripting (XSS)
Severity: High
Vulnerable URL: /search?q=<script>alert(1)</script>
Remediation: Sanitize inputs using DOMPurify or apply strict server-side input validation.
References:
  • OWASP: XSS Overview
  • PortSwigger: XSS Academy
  • DOMPurify GitHub
Vulnerability: Open Redirect
Severity: Medium
Vulnerable URL: /redirect?url=http://malicious.com
Remediation: Validate redirect targets against a whitelist and enforce domain-level restrictions.
References:
  • OWASP: Open Redirects
  • PortSwigger: Open Redirect
Vulnerability: Reusable Promo Codes
Severity: Logic-Level
Vulnerable URL: /apply-coupon?code=WELCOME100
Remediation: Invalidate coupon post-use and enforce ownership validation using session/user ID.
References:
  • PortSwigger: Business Logic Vulnerabilities
  • OWASP: Enforcing Access Controls
