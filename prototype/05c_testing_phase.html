Scan Complete — 3 issues detected:

🛑 XSS in Search Field

Risk: Medium

URL: https://yourapp.com/search?q=<script>alert(1)</script>

Issue:

Search results reflect unsanitized user input directly into the DOM.

Recommended Mitigation:
  • Sanitize user input using DOMPurify or templating auto-escapes
  • Apply Content Security Policy (CSP)
🔁 Open Redirect

Risk: High

URL: https://yourapp.com/redirect?next=http://evil.com

Issue:

Unvalidated redirection parameter allows phishing attacks.

Recommended Mitigation:
  • Whitelist internal redirect paths
  • Validate query parameters before using them in redirects
💸 Reusable Coupons

Risk: Critical

URL: https://yourapp.com/apply-coupon?code=DISCOUNT50

Issue:

Business logic flaw: Coupons can be reused indefinitely due to lack of server-side tracking.

Recommended Mitigation:
  • Log each coupon usage and enforce limits
  • Use signed one-time tokens for coupons
